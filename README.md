# restaurantList
## 介紹

創建屬於自己的餐廳清單

## 功能

- 查看所有餐廳清單
- 可查看每家餐廳詳細資料
- 搜尋餐廳

## 開始使用

1. 需安裝node.js 與 npm
2. 在終端機使用 git clone 複製 GitHub 的專案到本機端
    
    ```jsx
    git clone https://github.com/pierceshih15/restaurantList.git
    ```
    
3. 透過終端機進入專案資料夾
    
    ```jsx
    cd restaurant_list
    ```
    
4. 安裝 npm套件，在終端機輸入：
    
    ```jsx
    npm install
    ```
    
5. 使用 nodemon 套件連結至伺服器，在終端機輸入：
    
    ```jsx
    nodemon app.js
    ```
    
6. 欲中斷連線，在終端機輸入：
    
    ```jsx
    ctrl + c
    ```
    

## 開發工具

- Node.js 14.16.0
- Express 4.18.1
- Express-Handlebars 3.0.0
- Bootstrap 5.2
- Font-awesome 6.1.2
